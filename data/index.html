Grida Google Fonts Archive

Welcome to the Google Fonts archive bucket by Grida ( grida.co), an open design tool.

Usage

To access a font preview, use the following path format:

/{id}.svg

Examples:

  • ./inter.svg
  • ./roboto.svg
  • ./poppins.svg

Metadata

To access metadata listings, use the JSON paths below:

  • ./google-fonts-v2.min.json
  • ./google-fonts-v2.json

About Grida

Grida is an open-source design tool focused on accessible and flexible design workflows. Visit the project on GitHub: github.com/gridaco/grida.
