# Popular Fonts API

API endpoint serving font popularity statistics from Google Fonts Analytics.

## Base URL

```
https://fonts.grida.co/api/popular
```

## Endpoint

### GET /api/popular

Returns a list of popular fonts sorted by popularity metrics.

#### Query Parameters

- `limit` (optional, default: `100`) - Number of fonts to return
- `sortBy` (optional, default: `rate`) - Field to sort by. Options:
  - `rate` - Popularity rate (default)
  - `total_views` - Total number of views
  - `year_views` - Views in the past year

#### Response

```json
{
  "fonts": [
    {
      "family": "Roboto",
      "rate": 127147.02,
      "total_views": 32056497037737,
      "year_views": 3551116049868,
      "year_change": 0.09
    }
  ],
  "total": 1916,
  "limit": 100,
  "sortBy": "rate"
}
```

#### Response Fields

- `fonts` - Array of font statistics objects
  - `family` - Font family name
  - `rate` - Popularity rate
  - `total_views` - Total number of views
  - `year_views` - Views in the past year
  - `year_change` - Year-over-year change percentage (e.g., 0.09 = 9% increase)
- `total` - Total number of fonts available
- `limit` - Number of fonts returned (after applying limit parameter)
- `sortBy` - Field used for sorting

#### Examples

```bash
# Get top 100 fonts by rate (default)
curl https://fonts.grida.co/api/popular

# Get top 50 fonts sorted by total views
curl "https://fonts.grida.co/api/popular?limit=50&sortBy=total_views"

# Get top 10 fonts by year views
curl "https://fonts.grida.co/api/popular?limit=10&sortBy=year_views"
```

---

## Internal: Generating stats.json

The `stats.json` file is generated by running the `google_fonts_metadata_stats.py` script:

```bash
uv run python tools/google_fonts_metadata_stats.py --output app/api/popular/stats.json
```

The script fetches data from `https://fonts.google.com/metadata/stats` (which serves the same data as the Google Fonts Analytics page) and extracts relevant fields.

Note: The Google Fonts metadata endpoint is non-official and may break at any time. This file should be manually updated periodically.
